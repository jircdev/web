# CSS

Es un lenguaje de estilo utilizado para describir la apariencia de un documento HTML. Aquí están algunos de los conceptos básicos que se deben aprender sobre CSS:

1. Selectores.
    - de etiqueta.
    - de clases, por ejemplo `.class-name` que equivale al atributo `class` en HTML o `className` desde JavaScript.
    - id: `#id-name`.
    - Selectores de atributos: `input[type="text"]`..
    - Selector Universal "\*".
    - Selectores descendientes, hermanos
    - Jerarquía.
2. Propiedades y custom properties.
3. Unidades de medida.
4. Valores de color: Hexadecimal, HSL, rgb y rgba.
5. Box Model:
    - contenido.
    - Paddings
    - Margins
    - borders
    - Opacidad.
    - Shadows.
    - Transiciones.
6. Layout: Posicionamiento y flujo de elementos en la web. Uso de : `display`, `float`, `flexbox`, `grid`, entre otros.

7. media-queries.
8. Flexbox en detalle.
9. Grid en detalle.
10. Tipografia.
11. Animaciones.
